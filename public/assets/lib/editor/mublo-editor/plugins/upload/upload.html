MubloEditor - 이미지 업로드 플러그인

테스트 방법:
  1. PHP 서버 실행 (프로젝트 루트): php -S localhost:8000
  2. 브라우저 접속: http://localhost:8000/plugins/upload/upload.html
  3. 이미지 파일을 에디터에 드래그앤드롭, 붙여넣기, 또는 이미지 버튼 클릭

가장 간단한 방법 - uploadUrl만 설정하면 자동으로 업로드됩니다.
