# 게시판 운영

Board 패키지가 제공하는 게시판 시스템의 운영 가이드입니다.

## 게시판 생성

### 그룹 만들기

관리자 > 게시판 > 게시판 그룹

게시판을 묶는 상위 단위입니다. 그룹 안에 여러 게시판을 배치합니다.

| 항목 | 설명 |
|------|------|
| 그룹명 | "일반", "커뮤니티" 등 |
| 슬러그 | URL에 사용되는 영문 식별자 |
| 정렬 순서 | 목록에서의 표시 순서 |

### 게시판 추가

관리자 > 게시판 > 게시판 설정 > 추가

| 항목 | 설명 |
|------|------|
| 게시판명 | "공지사항", "자유게시판" 등 |
| 슬러그 | URL 식별자 (`/board/{slug}`) |
| 그룹 | 소속 그룹 선택 |
| 스킨 | basic, gallery 등 |
| 에디터 | 글 작성 에디터 (기본: mublo-editor) |

설치 시 기본으로 **공지사항** (스태프만 글쓰기)과 **자유게시판** (회원 글쓰기)이 생성됩니다.

## 게시판 설정

### 기본 설정

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 스킨 | 게시판 디자인 | basic |
| 에디터 | 글 작성 에디터 | mublo-editor |
| 페이지당 글 수 | 목록에 표시할 글 개수 | 15 |
| 공지 표시 수 | 상단 고정 공지 개수 | 5 |

### 기능 토글

| 기능 | 설명 | 기본 |
|------|------|------|
| 댓글 | 댓글 작성 허용 | 활성 |
| 리액션 | 좋아요 등 반응 버튼 | 활성 |
| 파일 첨부 | 파일 업로드 | 활성 |
| 링크 | 외부 링크 첨부 (OG 메타 추출) | 활성 |
| 카테고리 | 글 분류 | 비활성 |

### 파일 업로드 제한

| 항목 | 기본값 |
|------|--------|
| 최대 파일 수 | 2개 |
| 최대 파일 크기 | 2MB |
| 허용 확장자 | jpg, jpeg, png, gif, pdf, zip, hwp, doc, docx, xls, xlsx, ppt, pptx |

### 리액션 설정

기본 리액션은 "좋아요"(👍)입니다. 게시판별로 리액션 타입을 커스터마이징할 수 있습니다 (라벨, 아이콘, 색상).

## 권한 설정

4단계 계층 구조로 권한이 적용됩니다. 하위 설정이 상위를 덮어씁니다.

```
그룹 권한 (가장 넓은 범위)
  → 게시판 권한 (게시판별 재정의)
    → 카테고리 매핑 권한 (카테고리별 재정의)
      → 개별 글 권한 (글별 재정의)
```

### 권한 종류

| 권한 | 설명 |
|------|------|
| 목록 | 글 목록 조회 레벨 |
| 읽기 | 글 내용 조회 레벨 |
| 쓰기 | 글 작성 레벨 |
| 댓글 | 댓글 작성 레벨 |
| 다운로드 | 첨부파일 다운로드 레벨 |

### 레벨 값

| 값 | 의미 |
|----|------|
| 0 | 비회원 허용 |
| 1 | 일반 회원 |
| 230 | 스태프 |
| 255 | 최고 관리자 |

예시: 글쓰기 레벨을 230으로 설정하면 스태프 이상만 글을 작성할 수 있습니다.

### 비밀글 / 비밀게시판

| 기능 | 설명 |
|------|------|
| **비밀글** | 개별 글을 비밀로 설정. 작성자와 관리자만 열람 가능 |
| **비밀게시판** | 게시판 전체를 비밀 모드로. 모든 글이 자동으로 비밀글 (1:1 문의용) |

## 카테고리

### 카테고리 생성

관리자 > 게시판 > 카테고리

카테고리는 게시판과 **다대다(M:N)** 관계입니다. 하나의 카테고리를 여러 게시판에서 재사용할 수 있습니다.

### 게시판에 연결

게시판 설정에서 카테고리를 선택하여 연결합니다.

### 카테고리별 권한

카테고리 매핑 단위로 읽기/쓰기 등 권한을 재정의할 수 있습니다.

## 게시글 관리

관리자 > 게시판 > 게시글 관리

### 글 상태

| 상태 | 설명 |
|------|------|
| **게시됨** (published) | 정상 공개 |
| **임시저장** (draft) | 비공개 임시 저장 |
| **삭제됨** (deleted) | 소프트 삭제 |

### 일괄 작업

목록에서 글을 선택하여 일괄 삭제/상태 변경이 가능합니다.

### 공지사항

글의 `is_notice` 플래그를 활성화하면 목록 상단에 고정됩니다.

## 댓글

- **중첩 댓글** — 댓글에 답글 가능 (depth 기반 계층)
- **비밀 댓글** — 작성자와 글 작성자만 열람
- **비회원 댓글** — 비회원 허용 게시판에서 이름/비밀번호 입력 후 작성

## 포인트 설정

관리자 > 게시판 > 포인트 설정 (MemberPoint 플러그인 활성화 시)

| 항목 | 설명 |
|------|------|
| 글쓰기 포인트 | 글 작성 시 지급 |
| 댓글 포인트 | 댓글 작성 시 지급 |
| 조회 포인트 | 글 조회 시 차감 (선택) |
| 다운로드 포인트 | 파일 다운로드 시 차감 (선택) |

게시판별 또는 그룹별로 다른 포인트 정책을 설정할 수 있습니다.

### 일일 작성 제한

회원 레벨별로 하루 작성 가능한 글/댓글 수를 제한할 수 있습니다.

```json
{"0": 5, "1": 20, "230": null}
```
- 비회원: 5개, 일반 회원: 20개, 스태프: 무제한

## 커뮤니티 기능

Board 패키지는 커뮤니티 허브 페이지를 제공합니다.

- `/community` — 전체 피드
- `/community/popular` — 인기글 (조회수/반응 기반)
- `/community/group/{slug}` — 그룹별 필터
