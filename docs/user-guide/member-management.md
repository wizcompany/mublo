# 회원 관리

## 회원 목록

관리자 > 회원관리 > 회원목록

회원 목록에서 검색, 필터링, 상세 보기가 가능합니다.

### 검색 조건
- 아이디, 닉네임, 이메일
- 회원 등급
- 상태 (활성/비활성/휴면/차단)
- 가입일 범위

## 회원 등급

관리자 > 회원관리 > 회원등급

### 기본 등급 (설치 시 생성)

| 등급명 | 레벨값 | 설명 |
|--------|--------|------|
| SUPER | 255 | 최고 관리자 (`is_super=1`) |
| STAFF | 230 | 운영 스태프 (`is_admin=1`) |
| PARTNER | 220 | 파트너 |
| SELLER | 215 | 판매자 |
| SUPPLIER | 210 | 공급자 |
| BASIC | 1 | 일반 회원 |

### 등급 생성

| 항목 | 설명 |
|------|------|
| 등급명 | 표시 이름 |
| 레벨값 | 1~255 (높을수록 상위) |
| 관리자 여부 | `is_admin` 플래그 (관리자 메뉴 접근 가능) |
| Super 여부 | `is_super` 플래그 (모든 권한 통과) |

레벨값은 게시판 권한, 페이지 접근 제한 등에서 기준으로 사용됩니다.

## 회원 정보 관리

### 회원 등록 (관리자)

관리자가 직접 회원을 등록할 수 있습니다. 등록 시 `MemberRegisteredByAdminEvent`가 발행되어 포인트 지급 등 후속 처리가 실행됩니다.

### 회원 수정

- 기본 정보 (닉네임, 이메일 등)
- 등급 변경 (레벨 변경 시 포인트 지급 이벤트 발생)
- 상태 변경
- 커스텀 필드 값 수정

### 회원 상태

| 상태 | 설명 |
|------|------|
| **active** | 정상 이용 가능 |
| **inactive** | 비활성 (로그인 차단) |
| **dormant** | 휴면 (장기 미접속) |
| **blocked** | 차단 (관리자 처분) |

## 포인트

MemberPoint 플러그인이 활성화되면 포인트 기능을 사용할 수 있습니다.

### 포인트 조회

관리자 > 회원포인트 > 포인트 내역

도메인별 전체 포인트 이력을 조회합니다. 회원별, 기간별, 출처별 필터링이 가능합니다.

### 수동 지급/차감

관리자 > 포인트 관리 > 수동 조정

| 항목 | 설명 |
|------|------|
| 대상 회원 | 회원 검색/선택 |
| 금액 | 양수(지급) / 음수(차감) |
| 사유 | 관리자 메모 |

차감 시 잔액이 부족하면 실패합니다.

### 자동 지급 규칙

| 이벤트 | 설명 |
|--------|------|
| 회원가입 | 가입 시 자동 포인트 지급 |
| 등급 변경 | 레벨 상승 시 추가 포인트 |
| 글쓰기/댓글 | 게시판 포인트 설정에 따라 |
| 조회/다운로드 | 게시판 포인트 설정에 따라 (차감) |

### 무결성 검증

BalanceManager가 원장(balance_logs)과 스냅샷(members.point_balance)의 일치 여부를 검증합니다. 불일치 시 관리자가 복구할 수 있습니다.

## 관리자 권한

관리자 > 환경설정 > 권한관리

### 메뉴별 접근 권한

Super Admin 외의 관리자는 메뉴별로 접근을 제한할 수 있습니다.

- **Negative 모델** — 기본 허용, 거부할 메뉴를 선택
- 등급별로 거부 메뉴 설정 가능
- AJAX 요청 시 거부된 메뉴는 JSON 403 반환

## 커스텀 필드

관리자 > 회원관리 > 커스텀 필드

회원 정보에 추가 필드를 만들 수 있습니다.

지원 타입:
- 텍스트, 텍스트영역
- 숫자
- 날짜
- 선택 (드롭다운, 라디오)
- 체크박스
- 파일 업로드
- 주소 (우편번호 검색)

회원가입 폼과 관리자 회원 수정 화면에 자동으로 표시됩니다.
