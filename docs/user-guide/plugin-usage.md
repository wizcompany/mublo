# 플러그인 사용법

> 이 문서는 현재 공개 저장소에 포함된 플러그인만 설명합니다.
> 기본 포함 플러그인은 `Banner`, `Faq`, `MemberPoint`, `Popup`, `SnsLogin`, `Survey`, `VisitorStats`, `Widget`입니다.

## 플러그인 설치/활성화

관리자 > 시스템 > 확장 관리

활성화할 플러그인을 선택하면 해당 도메인에서 플러그인 기능이 활성화됩니다. 도메인별로 다른 플러그인 조합을 사용할 수 있습니다.

## Banner (배너)

관리자 > 배너 관리

### 배너 등록

| 항목 | 설명 |
|------|------|
| 제목 | 관리자 식별용 |
| PC 이미지 | 데스크톱 표시 이미지 |
| 모바일 이미지 | 모바일 표시 이미지 (선택) |
| 링크 URL | 클릭 시 이동 URL |
| 링크 타겟 | 새 창(_blank) / 현재 창(_self) |
| 표시 시작일 | 스케줄 표시 시작 |
| 표시 종료일 | 스케줄 표시 종료 |

### 블록에 배치

블록 편집에서 콘텐츠 타입을 "배너"로 선택하고, 표시할 배너를 선택합니다. 리스트 또는 슬라이드 모드로 표시할 수 있습니다.

## FAQ

관리자 > FAQ 관리

### 카테고리 관리

FAQ 항목을 분류하는 카테고리를 만듭니다. 카테고리별 슬러그와 정렬 순서를 설정합니다.

### FAQ 항목 등록

| 항목 | 설명 |
|------|------|
| 카테고리 | 소속 카테고리 |
| 질문 | FAQ 질문 텍스트 |
| 답변 | FAQ 답변 (에디터) |
| 정렬 순서 | 표시 순서 |

### 프론트 페이지

활성화 시 `/faq` 페이지가 자동 생성됩니다. 카테고리별로 접이식(아코디언) 형태로 표시됩니다.

### 블록에 배치

블록 편집에서 "FAQ" 콘텐츠 타입을 선택하면 특정 카테고리의 FAQ를 블록에 표시할 수 있습니다.

## Widget (위젯)

관리자 > 위젯 관리

### 위젯 항목 등록

| 항목 | 설명 |
|------|------|
| 아이콘 | 아이콘 이미지 또는 클래스 |
| 링크 URL | 클릭 시 이동 URL |
| 표시 디바이스 | PC만 / 모바일만 / 전체 |

### 표시 위치

- **PC** — 화면 측면 고정 사이드바
- **모바일** — 화면 하단 고정 아이콘 바

## Popup (팝업)

관리자 > 팝업 관리

### 팝업 생성

| 항목 | 설명 |
|------|------|
| 제목 | 관리자 식별용 |
| 내용 | HTML 에디터로 작성 |
| 표시 시작일/종료일 | 스케줄 설정 |
| 디바이스 | PC/모바일/전체 |
| 빈도 | 매번/세션당 1회/하루 1회 |

레이어 팝업으로 표시되며, 사용자가 닫기 버튼으로 닫을 수 있습니다.

## VisitorStats (방문자 통계)

관리자 > 방문자 통계

### 통계 대시보드

서버사이드 방문자 추적으로, 외부 서비스(GA) 없이 자체 통계를 제공합니다.

### 제공 통계

| 리포트 | 설명 |
|--------|------|
| 날짜별 | 일별 방문자/페이지뷰 |
| 시간별 | 시간대별 트래픽 분포 |
| 페이지별 | 인기 페이지 순위 |
| 유입경로별 | 리퍼러 분석 |
| 캠페인별 | UTM 파라미터 기반 캠페인 추적 |
| 전환 추적 | 폼 제출 등 전환 이벤트 |

### 대시보드 위젯

활성화 시 관리자 대시보드에 방문자 통계 위젯이 추가됩니다.

## SnsLogin (소셜 로그인)

관리자 > 소셜 로그인 설정

### 지원 제공자

| 제공자 | 설정 필요 |
|--------|----------|
| 네이버 | Client ID, Client Secret |
| 카카오 | REST API 키, Redirect URI |
| 구글 | Client ID, Client Secret |

### 설정 방법

1. 각 제공자의 개발자 센터에서 앱 등록
2. Redirect URI 설정: `https://your-domain.com/sns-login/callback/{provider}`
3. 관리자 > 소셜 로그인 설정에서 API 키 입력

### 동작 방식

- 로그인 폼에 소셜 로그인 버튼이 자동 추가됩니다
- 최초 소셜 로그인 시 자동 회원가입 (설정에 따라 추가 정보 입력 필요)
- 기존 회원과 소셜 계정 연결/해제 가능

## MemberPoint (포인트)

관리자 > 회원포인트

### 자동 지급 규칙

| 이벤트 | 설명 |
|--------|------|
| 회원가입 포인트 | 가입 시 자동 지급 |
| 등급 변경 포인트 | 등급 상승 시 지급 |

게시판 관련 포인트는 Board 패키지의 포인트 설정에서 관리합니다.

### 포인트 내역

회원별/기간별 포인트 이력을 조회할 수 있습니다.

## Survey (설문)

관리자 > 설문 관리

### 설문 작성

| 항목 | 설명 |
|------|------|
| 설문 제목 | 표시 제목 |
| 설문 설명 | 안내 문구 |
| 시작일/종료일 | 참여 기간 |
| 질문 목록 | 다양한 질문 타입 |

### 질문 타입

텍스트, 객관식(단일 선택), 객관식(복수 선택), 별점 등

### 결과 확인

관리자에서 설문 결과를 집계하여 확인할 수 있습니다.

### 블록 연동

블록 편집에서 "설문" 콘텐츠 타입으로 참여 폼을 페이지에 배치할 수 있습니다.

## 공개 저장소 기준 참고

- 이 문서에 없는 플러그인은 현재 공개 저장소 기본 범위에 포함되지 않습니다.
- 블록 편집 화면에 표시되는 콘텐츠 타입은 활성화된 Package/Plugin 조합에 따라 달라질 수 있습니다.
