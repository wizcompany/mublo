# 관리자 화면 기본 사용법

## 관리자 메뉴 구조

관리자 화면은 왼쪽 사이드바 메뉴로 구성됩니다.

### Core 메뉴 (기본)

| 메뉴 | 하위 | 설명 |
|------|------|------|
| **대시보드** | — | 위젯 기반 현황 화면 |
| **환경설정** | 기본/회사/SEO/테마/검색 | 사이트 전체 설정 |
| **회원관리** | 회원목록/회원등급/커스텀필드 | 회원 CRUD, 레벨 관리 |
| **블록관리** | 블록페이지/블록행 | 페이지 빌더 |
| **메뉴관리** | 메인/푸터/유틸/마이페이지 | 네비게이션 메뉴 |
| **약관관리** | — | 이용약관/개인정보 등 |
| **도메인** | — | 멀티 도메인 관리 |
| **시스템** | 시스템정보/확장관리 | 서버 상태, 플러그인/패키지 활성화 |

### Package/Plugin 메뉴

활성화된 패키지/플러그인이 자동으로 메뉴에 추가됩니다.

- **Board** — 게시판설정, 게시판그룹, 게시글관리, 카테고리, 포인트
- **Banner** — 배너 관리
- **FAQ** — FAQ 관리
- 기타 활성화된 플러그인...

## 대시보드

로그인 후 첫 화면입니다. 위젯 기반으로 구성되며, 위젯을 드래그하여 배치를 변경할 수 있습니다.

기본 위젯:
- **시스템 정보** — PHP 버전, DB 상태, 디스크 사용량
- **회원 통계** — 최근 가입자, 전체 회원 수
- **최근 공지** — 공지사항 최신글 (Board 패키지 활성화 시)
- **방문자 통계** — 일별 방문자 (VisitorStats 플러그인 활성화 시)

## 공통 조작법

### 목록 화면

- **검색** — 상단 검색 필드로 필터링
- **정렬** — 컬럼 헤더 클릭으로 정렬 변경
- **페이지네이션** — 하단 페이지 이동
- **일괄 작업** — 체크박스 선택 후 일괄 삭제/상태 변경

### 등록/수정 화면

- **폼 제출** — 저장 버튼 클릭 (AJAX 처리)
- **필수 입력** — 빨간 별표(*) 표시 필드
- **이미지 업로드** — 파일 선택 또는 드래그 앤 드롭

### 권한

- **Super Admin** (레벨 255) — 모든 메뉴 접근 가능
- **일반 관리자** — 권한 설정에 따라 메뉴별 접근 제한
- 거부된 메뉴 접근 시 403 에러 표시
