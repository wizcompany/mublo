# 사용자 가이드

Mublo Framework를 설치하고 사이트를 운영하는 방법을 안내합니다.

> 이 가이드는 초기 공개 번들 기준으로 작성되었습니다.
> 기본 공개 범위는 `Board`, `Shop` 패키지와 `Banner`, `Faq`, `MemberPoint`, `Popup`, `SnsLogin`, `Survey`, `VisitorStats`, `Widget` 플러그인입니다.
> 현재 공개 저장소는 위 범위만을 기준으로 설명합니다.
