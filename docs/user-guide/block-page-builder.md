# 블록으로 페이지 만들기

## 블록이란?

블록 시스템은 코드 수정 없이 관리자 화면에서 페이지를 구성하는 페이지 빌더입니다. 행(Row)과 열(Column)을 조합하고, 각 열에 콘텐츠(HTML, 배너, 게시판 최신글 등)를 배치합니다.

```
┌─────────────────────────────────────────┐ ← 블록 페이지
│ ┌─────────────────────────────────────┐ │ ← 행 (Row)
│ │ ┌───────────┐ ┌───────────────────┐ │ │ ← 열 (Column)
│ │ │  배너      │ │  게시판 최신글     │ │ │ ← 콘텐츠 타입
│ │ └───────────┘ └───────────────────┘ │ │
│ └─────────────────────────────────────┘ │
│ ┌─────────────────────────────────────┐ │
│ │ ┌─────────────────────────────────┐ │ │
│ │ │       HTML 직접 입력             │ │ │
│ │ └─────────────────────────────────┘ │ │
│ └─────────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

## 블록 페이지 관리

관리자 > 블록관리 > 블록페이지

### 페이지 생성

| 항목 | 설명 |
|------|------|
| 페이지 코드 | 고유 식별자 (영문, URL에 사용) |
| 페이지 제목 | 관리자/브라우저에 표시 |
| 레이아웃 | 1단(전체), 2단 좌측, 2단 우측, 3단 |
| SEO 설정 | 메타 제목, 설명, 키워드 |
| 접근 레벨 | 비회원/회원/관리자 등 접근 제한 |
| 헤더/푸터 | 사용 여부 선택 |

### 레이아웃 타입

| 타입 | 설명 |
|------|------|
| **1단 (전체)** | 사이드바 없이 전체 너비 사용 |
| **2단 좌측** | 왼쪽 사이드바 + 콘텐츠 |
| **2단 우측** | 콘텐츠 + 오른쪽 사이드바 |
| **3단** | 왼쪽 사이드바 + 콘텐츠 + 오른쪽 사이드바 |

### 메뉴와 연결

메뉴 관리에서 메뉴 항목의 URL을 블록 페이지 코드(`/p/{pageCode}`)로 설정하면 해당 페이지가 표시됩니다.

## 행(Row) 구성

관리자 > 블록관리 > 블록행

### 행 추가

| 항목 | 설명 |
|------|------|
| 위치 | 메인화면, 좌측 사이드바, 우측 사이드바, 서브페이지 상단/하단, 콘텐츠 상단/하단 |
| 관리 제목 | 관리자 식별용 |
| 너비 | 와이드(100%) / 콘테이너(중앙 정렬) |
| 열 개수 | 1~6개 |
| 열 간격 | 열 사이 여백 (px) |
| 배경 | 색상, 이미지, 반복, 위치, 크기 |
| 패딩 | PC/모바일 별도 설정 |
| 높이 | PC/모바일 별도 설정 |

### 블록 위치

| 위치 | 설명 |
|------|------|
| **메인화면** | 메인 페이지 본문 |
| **왼쪽 사이드바** | 2단/3단 레이아웃의 왼쪽 |
| **오른쪽 사이드바** | 2단/3단 레이아웃의 오른쪽 |
| **서브페이지 상단** | 서브페이지 콘텐츠 위 |
| **서브페이지 하단** | 서브페이지 콘텐츠 아래 |
| **콘텐츠 상단** | 본문 콘텐츠 바로 위 |
| **콘텐츠 하단** | 본문 콘텐츠 바로 아래 |

## 열(Column)과 콘텐츠

행 편집 화면에서 각 열의 콘텐츠를 설정합니다.

### 열 설정

- **너비 비율** — 열의 가로 비율 (%, px)
- **패딩** — PC/모바일 별도
- **배경** — 색상, 이미지
- **테두리** — 스타일, 색상, 모서리 둥글기

### 콘텐츠 타입

#### Core 타입

| 타입 | 설명 |
|------|------|
| **HTML 직접입력** | 에디터로 자유롭게 HTML 작성 |
| **이미지** | 이미지 업로드 + 링크 설정 |
| **동영상** | YouTube, Vimeo 등 임베드 |
| **로그인 위젯** | 로그인/로그아웃 상태 표시 |
| **메뉴** | 등록된 메뉴를 블록으로 표시 |
| **PHP 파일 포함** | 지정한 PHP 파일을 인클루드 (개발자용) |

#### Package/Plugin 타입 (활성화 시)

| 타입 | 제공자 | 설명 |
|------|--------|------|
| **게시판 최신글** | Board | 특정 게시판의 최신 글 표시 |
| **게시판 그룹** | Board | 여러 게시판을 탭/그리드로 표시 |
| **배너** | Banner 플러그인 | 배너 슬라이드/리스트 |
| **FAQ** | FAQ 플러그인 | FAQ 아코디언 표시 |
| **설문** | Survey 플러그인 | 설문 참여 폼 |

현재 공개 저장소 기준으로 문서화된 확장 타입은 위 범위까지입니다. 실제 표시 가능한 타입은 활성화된 Package/Plugin 조합에 따라 달라집니다.

### 디스플레이 스타일

콘텐츠 타입이 `hasStyle`을 지원하면 표시 방식을 선택할 수 있습니다.

| 스타일 | 설명 |
|--------|------|
| **list** | 세로 목록 |
| **slide** | 가로 슬라이드 (Swiper) |

PC/모바일 별도 설정: 표시 개수, 컬럼 수, 자동 재생, 반복

## 스킨 선택

각 콘텐츠 타입은 여러 스킨을 가질 수 있습니다. 열 설정에서 스킨을 선택하면 해당 디자인으로 렌더링됩니다.

기본 스킨은 `basic`이며, 추가 스킨을 만들어 적용할 수 있습니다.

## 제목 설정

각 블록 열에 제목을 표시할 수 있습니다.

- 제목 텍스트, 색상, 크기 (PC/모바일 별도)
- 제목 이미지 (PC/모바일 별도)
- "더보기" 링크 URL
- 부제목 (copytext)

## 캐싱

블록은 자동으로 캐싱됩니다. 블록 내용을 수정하면 관련 캐시가 자동으로 무효화됩니다. 로그인 위젯처럼 사용자별로 다른 콘텐츠는 캐싱되지 않습니다.
